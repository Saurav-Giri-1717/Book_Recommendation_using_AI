
# Book Recommendation System using AI

## Files Included:
- `book_recommendation_ai.py`: Python script to recommend books based on content similarity.
- `books.csv`: Sample dataset with book metadata (title, author, genre, description).
- `README.txt`: Project overview and usage instructions.

## How to Use:
1. Ensure Python is installed with the required libraries:
   ```
   pip install pandas scikit-learn
   ```
2. Run the script in your terminal or VS Code:
   ```
   python book_recommendation_ai.py
   ```
3. You can replace `books.csv` with a larger dataset for better recommendations.

## Dataset Format:
The CSV file should contain the following columns:
- `title`
- `author`
- `genre`
- `description`
